# Next.js Boilerplate

プロジェクトの構成の概要です。

- Next.js
- TypeScript
- ESLint
- Prettier

ビルド設定のカスタマイズは`next.config.js`を編集してください。

## ディレクトリ

- `src`
  - 開発コードを入れておく。
- `public`
  - 静的ファイルを入れておく。

## コマンド

- `yarn dev`
  - `localhost:3000` でローカル環境が立ち上がります。
- `yarn build`
  - ビルドします。
- `yarn build:analyze`
  - ビルド時にBundle Analyzerを起動します。
- `yarn start`
  - 本番モードでアプリケーションを起動します。

その他は `package.json` をご参照ください。
